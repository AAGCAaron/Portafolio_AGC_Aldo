# Portafolio Profesional - Aldo Gutierrez

Este proyecto es un portafolio web moderno y responsivo desarrollado con React y Vanilla CSS.

## Tecnologías Utilizadas
- React 19
- Vite
- Vanilla CSS (Variables CSS, Flexbox, Grid)
- React Icons

## Características Principales
- **Diseño Premium**: Paleta de colores Dark Mode.
- **Navegación Flotante**: Menú estilo "Píldora" con smooth scroll.
- **Fondo Animado**: Partículas interactivas en la sección Hero.
- **Timeline de Experiencia**: Visualización clara de la trayectoria profesional.
- **Grid Bento**: Sección "Sobre mí" con diseño modular.

## Instalación y Ejecución

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/AAGCAaron/Portafolio_AGC_Aldo.git
   ```

2. Instalar dependencias:
   ```bash
   npm install
   ```

3. Ejecutar servidor de desarrollo:
   ```bash
   npm run dev
   ```
